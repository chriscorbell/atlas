n8n lets you automate tasks across services using triggers, schedules, and APIs.

## Docker Compose
Create: `~/docker/compose/automation/n8n.compose.yml`

```yaml
services:
  n8n:
    image: n8nio/n8n:latest
    container_name: n8n
    ports:
      - "5678:5678"
    environment:
      - TZ=America/New_York
      - NODE_ENV=production
    volumes:
      - ~/docker/appdata/n8n:/home/node/.n8n
    restart: unless-stopped
    networks: [proxy]

networks:
  proxy:
    external: true
```

Start:
```bash
docker compose -f ~/docker/compose/automation/n8n.compose.yml up -d
```

Open:
- `http://<debian-ip>:5678`
